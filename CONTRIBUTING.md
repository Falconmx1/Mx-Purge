# Contribuir a Mx-Purge

¡Gracias por tu interés! 🎉

## ¿Cómo contribuir?

1. **Fork** el repositorio
2. **Crea una rama**: `git checkout -b feature/nueva-funcion`
3. **Haz tus cambios** y añade tests
4. **Asegura** que todo funciona: `python -m pytest tests/`
5. **Commit** con mensaje claro: `git commit -m "Añade: limpieza de navegadores"`
6. **Push** a tu fork: `git push origin feature/nueva-funcion`
7. **Abre un Pull Request**

## Reportar bugs

Usa el template de Issues en GitHub.
